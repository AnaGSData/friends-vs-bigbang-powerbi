# Friends vs The Big Bang Theory Power BI Dashboard

Power BI pair project comparing **Friends** and **The Big Bang Theory** through ratings, audience metrics and visual storytelling.

This project was developed during the **Adalab Data Analytics & AI Bootcamp** as a pair visualization project.

---

## Overview

The goal of this project was to compare two highly successful sitcoms, **Friends** and **The Big Bang Theory**, using Power BI dashboards to analyze ratings, audience trends, episode performance and season-level patterns.

The project focuses on transforming entertainment data into clear visual insights, applying dashboard design, business intelligence and storytelling techniques.

---

## Project Context

This was a **pair project** developed during the Adalab Data Analytics & AI Bootcamp.

The project simulates a real data visualization scenario where a stakeholder needs to compare the performance of two series and support a data-driven decision through interactive dashboards.

---

## Business Scenario

The analysis is framed as a decision-making case for a streaming platform that needs to evaluate the performance of two sitcoms.

The dashboard helps compare both series through metrics such as ratings, audience evolution, episode performance and consistency across seasons.

---

## Objective

The main objective was to design a Power BI dashboard capable of answering questions such as:

- Which series shows stronger rating performance?
- How do ratings evolve across seasons?
- Which episodes stand out the most?
- How does audience behavior differ between both series?
- Which series shows greater consistency over time?
- What insights can support a data-driven content decision?
